# 📎 文具領取 · 高雄特教

這是一個專為 **高雄市立高雄特殊教育學校** 設計的簡易文具領取網頁。它將傳統的紙本「物品收發與盤存月報表」數位化，提供一個直觀、便於在手機上操作的工具，讓物品管理更加即時與便利。

👉 **線上使用**：[https://cheap-labor782.github.io/Stationery/](https://cheap-labor782.github.io/Stationery/)

## ✨ 功能特點

*   **完整品項清單**：內建 49 種常用文具與辦公用品，品名、規格、單位與即時庫存一目瞭然。
*   **直觀數量調整**：每個品項都配備「+」與「-」按鈕，方便您快速調整欲領取的數量。
*   **一鍵領取與即時回饋**：點擊「領取」按鈕即可完成扣減庫存，並有明確的提示訊息告知操作結果。
*   **按錯防護機制**：
    *   領取前會跳出確認對話框，防止誤觸。
    *   提供 **「復原」** 功能，可撤銷**上一次**的領取動作，將庫存恢復原狀。
*   **即時庫存總覽**：頁面底部固定顯示所有品項的「本月結存」總數，方便快速掌握整體庫存狀況。
*   **手機介面優化**：採用卡片式設計與適合觸控的操作按鈕，在行動裝置上使用體驗流暢。

## 🛠️ 技術細節

*   **純前端應用**：使用原生 HTML、CSS 與 JavaScript 開發，無需後端伺服器或資料庫。
*   **資料儲存**：所有品項與庫存數據以 JavaScript 陣列形式儲存於前端，重啟頁面即恢復初始狀態。
*   **核心邏輯**：
    *   庫存異動（領取/復原）即時更新 DOM 與總量顯示。
    *   透過 `lastAction` 變數記錄上一次的領取動作，以實現單次復原功能。
